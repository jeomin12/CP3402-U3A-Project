Goal/s

The goal of this project was to develop a fully functional, responsive WordPress website for U3A Townsville using a custom child theme. The website aims to provide information about membership, events, news, and community support while reflecting a clean, modern, and accessible design. The deployment was automated via GitHub Actions to a publicly accessible AWS Lightsail instance, with all theme files under version control. Success was measured by fully automated deployment, a visible live theme on the server, and functional page navigation.
